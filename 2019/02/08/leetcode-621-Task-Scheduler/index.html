题目

给一些任务tasks需要尽快完成，但是有一个限制，相同任务必须间隔n个其他任务。

由于”A”是次数最多的任务，它会把任务序列分割成 A X X A X X A X X，因此每次迭代就是以次数最多的任务来分割，这样tasks完成时间最小。

普通情况
